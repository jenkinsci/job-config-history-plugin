Introduzca el directorio donde se guardarán los registros de cambios. Esta información se debe introducir para evitar perder dichos registros cuando una ejecución se borra. Cuando no se selecciona esta opción, el registro de cambios se pierde cuando se borra una ejecución.

Se puede especificar bien una ruta absoluta o relativa. Cuando la ruta es relativa, el directorio se creará a partir del directorio HUDSON_ROOT
